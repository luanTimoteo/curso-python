# 📚 Aula 02 - Manipulação de Strings e Formatação

Nesta aula aprendi a trabalhar com textos e formatação de dados em Python.

## 🛠️ O que aprendi:
- **Métodos de Texto:** `.lower()`, `.upper()`, `.title()`, `.strip()` e `.replace()`
- **Análise de Texto:** `len()` para contar caracteres e `.find()` para localizar posições
- **Corte de Strings (Slicing):** Extrair partes específicas de um texto usando índices `[início:fim]`
- **F-Strings:** Formatação avançada de textos, moedas (`R$`) e porcentagens (`%`)

## 🎯 Exercício Prático:
Criar um script que identifica automaticamente o primeiro nome do usuário e o domínio do seu e-mail para gerar uma mensagem personalizada de cadastro.
