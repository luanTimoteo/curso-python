# Curso de Python 2025 - Fundamentos 🐍

Repositório dedicado aos estudos de lógica de programação, estruturas de dados e sintaxe da linguagem Python, acompanhando as aulas e resolvendo os desafios propostos na playlist "Curso de Python 2025" da Hashtag Programação.

O objetivo deste projeto é consolidar os pilares fundamentais da programação, construindo uma base sólida para a resolução de problemas, manipulação de dados estruturados e modularização de código.

## 📌 Conteúdo Estudado

* **Ambiente e Sintaxe:** Instalação de ferramentas, uso de variáveis e operadores matemáticos básicos.
* **Manipulação de Dados Individuais:** Operações essenciais com Strings (textos) e coleta de dados via `input`.
* **Estruturas de Controle:** Tomada de decisão com condicionais (`if`, `elif`, `else`) e comparadores lógicos.
* **Coleções e Estruturas de Dados:** Armazenamento estruturado usando Listas, Dicionários (chave-valor) e Tuplas.
* **Automação e Repetição:** Otimização de tarefas repetitivas e varredura de dados com laços `for`.
* **Modularização e Reutilização:** Criação de funções personalizadas com parâmetros e importação de módulos/bibliotecas externas.

## 📂 Organização das Pastas

* `📂 Aula 00 - Instalacao e Configuracao/` -> Configuração do ambiente de desenvolvimento (Python e VS Code).
* `📂 Aula 01 - Variaveis e Operacoes/` -> Conceitos iniciais de variáveis e cálculos numéricos.
* `📂 Aula 02 - Textos e Strings/` -> Métodos de formatação e manipulação de textos.
* `📂 Aula 03 - Inputs e Listas/` -> Coleta dinâmica de dados e introdução a coleções mutáveis.
* `📂 Aula 04 - Condicionais/` -> Estruturas de decisão e lógica booleana.
* `📂 Aula 05 - Dicionarios/` -> Estruturação de dados eficientes através de mapeamento chave-valor.
* `📂 Aula 06 - For e Repeticao/` -> Automatização de rotinas e laços de repetição.
* `📂 Aula 07 - Funcoes e Parametros/` -> Criação de blocos de códigos reutilizáveis e escopo de variáveis.
* `📂 Aula 08 - Tuplas e Multiplas Respostas/` -> Manipulação de coleções imutáveis e funções com retornos múltiplos.
* `📂 Aula 09 - Modulos e Bibliotecas/` -> Importação de recursos externos para expansão das capacidades do script.

## 🛠️ Ferramentas Utilizadas
* **Python 3**
* **Visual Studio Code (VS Code)**
* Git e GitHub
